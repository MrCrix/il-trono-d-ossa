## Setup Notion Integration (da fare)

1. Vai su https://www.notion.so/my-integrations
2. Crea una nuova integrazione
3. Copia l'API token
4. Condividi il database Notion con l'integrazione
5. Salva il token in `.env`

## Scripts Disponibili

- `export_session.py` - Esporta note sessione in formato condivisibile
